# VFM-Guidance

This repository hosts the different experimental projects on VFM. Including:
* Flow Matching (FM): Simple_FM_date
* Variaional Flow Matching (VFM): Simple_VFM_date
* VFM + Classifier-free guidance (CFG): Simple_CFG_VFM_date
* VFM + Classifier guidance (CG): Simple_tClassifier_VFM_date

The Latex for the corresponding code can be found here: https://www.overleaf.com/read/pwcmpjvjqxrw#5f1016 (note that it currently only contains the reasoning behind VFM + CG, and will be updated).
